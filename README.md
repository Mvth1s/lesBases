# lesBases

Ce dépôt a pour objectif de me permettre de réapprendre et de retravailler les bases de différents langages de programmation tels que `Python`, `C`, `C++`, `JavaScript`, etc.

## Structure du projet

Chaque langage dispose de son propre dossier dédié. Les projets de réapprentissage des bases sont organisés en sous-dossiers au sein de leur dossier parent respectif.

### Exemple de structure

```
.
├── README.md
├── python
│   ├── README.md
│   ├── hello.py
│   └── projet_python_1
├── javascript
│   └── projet_1
└── ...
```

## Organisation des dossiers

Chaque dossier dédié à un langage contient son propre fichier `README.md`. Ce fichier permet de suivre :
- Ce qui a été étudié
- Ce qui reste à voir
- Les projets à venir

## Philosophie

> "Keep it simple"

L'objectif est de rester simple et efficace dans l'apprentissage et la pratique des bases.
